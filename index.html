Overview

LaysNet is a pure Python research prototype for graph-based classification. It uses deterministic projections, stage-wise quantization, semaphore routing, and Markov edge memory rather than neural backpropagation.

What it does

  • Encodes numeric input into symbolic graph stages.
  • Stores class-conditioned node and edge counts.
  • Predicts using smoothed log-probabilities.
  • Returns an explainable route and trace.

Why use it

  • No required NumPy, PyTorch, or scikit-learn for core use.
  • CPU-friendly and inspectable.
  • Works with your real project data and CSV datasets.
  • Supports both tabular and flattened image inputs.

Quick Start

Use the bundled Iris dataset and project helpers to run a real train/test workflow.

from laysnet import LaysNetClassifier
from laysnet.datasets import load_iris_dataset, stratified_train_test_split

X, y = load_iris_dataset()
X_train, X_test, y_train, y_test = stratified_train_test_split(X, y, seed=11)

model = LaysNetClassifier(
    steps=24,
    bins=11,
    seed=5,
    smoothing=0.5,
    transition_weight=1.7,
)
model.fit(X_train, y_train)
print("accuracy:", model.score(X_test, y_test))
print("prediction:", model.predict_one(X_test[0]))
print("trace:", model.trace_one(X_test[0]))
Use your real data

The project has a real bundled Iris dataset in laysnet/data/iris.csv. Replace this with your own dataset using load_csv_dataset for correct training and testing.

Model internals

LaysNet trains by:

  1. Standardizing input features if normalize=True.
  2. Projecting each vector with deterministic seed-based projection weights.
  3. Quantizing projection values into one of bins codes per stage.
  4. Storing class-labeled node and directed edge visit counts.
  5. Scoring routes using class prior, node evidence, and edge transition evidence.

During prediction, the model:

  1. Encodes the input vector with learned thresholds.
  2. Routes each encoded stage through the semaphore router.
  3. Computes smoothed log-scores for every candidate label.
  4. Selects the highest-scoring label and returns a prediction trace.

API Reference

Core API

  • LaysNetClassifier — core classifier implementation.
  • LaysNetEnsembleClassifier — average multiple graph models.
  • SequentialGraphNetwork — torch-like pipeline builder.
  • GraphEncoder, SemaphoreRouter, MarkovMemory — pipeline layers.

Key methods

  • model.fit(X, y)
  • model.predict(X)
  • model.predict_one(row)
  • model.score(X, y)
  • model.trace_one(row)
  • model.graph_summary()
  • model.export_graph()

Layer pipeline example

import laysnet.nn as lnn

model = lnn.SequentialGraphNetwork(
    lnn.GraphEncoder(steps=24, bins=11, seed=5),
    lnn.SemaphoreRouter(),
    lnn.MarkovMemory(smoothing=0.5, transition_weight=1.7),
)
model.fit(X_train, y_train)
output = model.forward(X_test[0])
print(output.label)
print(output.encoded)
print(output.route)

Train & Test

Split your data with the built-in helper and train using real CSV data from your project.

from laysnet.datasets import load_csv_dataset, stratified_train_test_split
from laysnet import LaysNetClassifier

X, y = load_csv_dataset("data/my_data.csv", label_column=-1, has_header=True)
X_train, X_test, y_train, y_test = stratified_train_test_split(X, y, test_ratio=0.3, seed=42)

model = LaysNetClassifier(
    steps=24,
    bins=11,
    seed=13,
    smoothing=0.5,
    transition_weight=1.7,
)
model.fit(X_train, y_train)
print("test accuracy:", model.score(X_test, y_test))
print("graph summary:", model.graph_summary())

For project-specific validation, use your dataset file and label column instead of synthetic examples.

Compare on Iris with the repo script:

python3 examples/compare_iris.py

data/animals/train/cat/*.jpg
 data/animals/train/dog/*.jpg
 data/animals/test/cat/*.jpg
 data/animals/test/dog/*.jpg

python3 examples/train_image_folder.py --data data/animals --image-size 16

The script converts images to grayscale, resizes them, flattens pixels, and passes them to LaysNet.
