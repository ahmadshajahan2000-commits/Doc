Quick Start

Run the bundled Iris workflow or swap in your own CSV dataset immediately.

Iris train/test example
from laysnet import LaysNetClassifier
from laysnet.datasets import load_iris_dataset, stratified_train_test_split

X, y = load_iris_dataset()
X_train, X_test, y_train, y_test = stratified_train_test_split(X, y, seed=11)

model = LaysNetClassifier(
    steps=24,
    bins=11,
    seed=5,
    smoothing=0.5,
    transition_weight=1.7,
)
model.fit(X_train, y_train)
print("accuracy:", model.score(X_test, y_test))
print("trace:", model.trace_one(X_test[0]))
Use your own project data

Replace load_iris_dataset with load_csv_dataset("data/my_data.csv", label_column=-1, has_header=True) to run this on your real dataset.

Internals

Understand the model flow from raw features to graph memory.

Encoder

Standardizes input features, then projects each input vector using deterministic seed-based weights.

Quantizer

Partitions projections into bins discrete codes using training thresholds.

Graph memory

Stores node counts, node-label counts, edge counts, and edge-label counts for route scoring.

Prediction trace

Each prediction returns an explainable trace with:

  • encoded stage codes
  • semaphore-routed graph nodes
  • class log-scores
  • Euler balance error

Math

The model uses deterministic projection, quantization, and smoothed log-probability scoring.

Encoding formulas

Projection: p_s = x · w_s

Code assignment: c_s = bisect_right(thresholds_s, p_s)

Each stage s maps a projected value to one of bins discrete symbols.

Scoring formulas

Label prior:

P(y) = (count(y) + α) / (N + αK)

Node probability:

P(node | y) = (count_y(node) + α) / (count(node) + αK)

Edge probability:

P(edge | y) = (count_y(edge) + α) / (count(edge) + αK)

The final log score is:

score(y) = log P(y)
  + node_weight * Σ log P(node | y)
  + transition_weight * Σ log P(edge | y)
Route diagnostics

The Euler balance error measures directed imbalance:

error = Σ |outdeg(node) - indeg(node)|

Lower values imply more balanced directed transitions along the predicted route.

API Reference

Core classes

  • LaysNetClassifier — main classifier.
  • LaysNetEnsembleClassifier — average multiple models.
  • SequentialGraphNetwork — torch-like pipeline API.

Pipeline layers

  • GraphEncoder
  • SemaphoreRouter
  • MarkovMemory
Layer API example
import laysnet.nn as lnn

model = lnn.SequentialGraphNetwork(
    lnn.GraphEncoder(steps=24, bins=11, seed=5),
    lnn.SemaphoreRouter(),
    lnn.MarkovMemory(smoothing=0.5, transition_weight=1.7),
)
model.fit(X_train, y_train)
output = model.forward(X_test[0])
print(output.label)
print(output.encoded)
print(output.route)

Train & Test

Use your actual CSV dataset and built-in split helpers rather than dummy data.

Real dataset workflow
from laysnet.datasets import load_csv_dataset, stratified_train_test_split
from laysnet import LaysNetClassifier

X, y = load_csv_dataset("data/my_data.csv", label_column=-1, has_header=True)
X_train, X_test, y_train, y_test = stratified_train_test_split(X, y, test_ratio=0.3, seed=42)

model = LaysNetClassifier(
    steps=24,
    bins=11,
    seed=13,
    smoothing=0.5,
    transition_weight=1.7,
)
model.fit(X_train, y_train)
print("test accuracy:", model.score(X_test, y_test))
print("graph summary:", model.graph_summary())

Use the repo compare script for a full benchmark table:

python3 examples/compare_iris.py

Google Colab

Run the package in Colab with no GPU required. Use the same real dataset workflow as local install.

Colab install & usage
!pip install git+https://github.com/ahmadshajahan2000-commits/LaysNet.git
!python -m laysnet

from laysnet.datasets import load_iris_dataset, stratified_train_test_split
from laysnet.nn import SequentialGraphNetwork, GraphEncoder, SemaphoreRouter, MarkovMemory

X, y = load_iris_dataset()
X_train, X_test, y_train, y_test = stratified_train_test_split(X, y, test_ratio=0.3, seed=11)

model = SequentialGraphNetwork(
    GraphEncoder(steps=24, bins=11, seed=5),
    SemaphoreRouter(),
    MarkovMemory(smoothing=0.5, transition_weight=1.7),
)
model.fit(X_train, y_train)
print("accuracy:", model.score(X_test, y_test))
